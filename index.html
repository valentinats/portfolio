Мои компетенции

  • HTML и CSS

    Создание структуры сайта: параграфы, разделы, заголовки, навигационные панели и пр. С помощью CSS описывается внешний вид веб-страницы — определяется макет сайта, используемые шрифты и цвета.

  • JavaScript

    Создание интерактивных и динамических элементов сайта. Попапы, меню-бургеры, «карусели», формы обратной связи и пр. Работа с основами JavaScript, включая DOM-дерево, асинхронные запросы и манипуляции данными на стороне клиента.

  • Работа с библиотеками и фреймворками

    Использование библиотеки React для стандартизации и оптимизации разработки. Умение использовать готовые решения, адаптировать их под нужды проекта и расширять их функциональность.

  • Разработка для мобильных устройств и адаптивная верстка

    Оптимальное отображение интерфейсов на устройствах с разным разрешением экранов. Адаптивный дизайн для взаимодействия с кнопками, текстом, изображениями через разные устройства.
